Quick Start Guide
  1. Add a Device: Go to Devices and add your ZKTeco K40 device with IP address and port.
  2. Add Employees: Create employee records with unique user IDs (1-65535).
  3. Sync Employees: Upload employees to the device or download existing users from device.
  4. Download Attendance: Fetch attendance events from the device.
  5. Generate Reports: View daily, weekly, or monthly attendance reports.
Development Mode: The system is currently running in MOCK mode with simulated data. To connect to a real device, set ZK_TEST_MODE=False in your .env file.
